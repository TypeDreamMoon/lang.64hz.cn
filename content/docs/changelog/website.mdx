## 2026-04-28

### 自动部署

- 新增 GitHub Actions 部署工作流，推送到 `main` / `master` 或手动触发时会执行 typecheck、build 和 SFTP 上传。
- 部署脚本统一到 `upload.js`，通过 `SFTP_HOST`、`SFTP_USER`、`SFTP_PASSWORD` / `SFTP_PRIVATE_KEY`、`SFTP_REMOTE_DIR` 等环境变量或 secrets 配置。
- 上传前会清理远端目录，同时保留 `.user.ini` 和 `.htaccess`。
